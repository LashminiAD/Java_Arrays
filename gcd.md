## Approaches to GCD
1. **Euclidean Algorithm**: 
   - Description: A classic method that finds GCD using division.
2. **Prime Factorization**: 
   - Description: Finds GCD by determining the prime factors of the numbers.
3. **Binary GCD Algorithm**: 
   - Description: A bitwise algorithm that is efficient for finding GCD.
4. **Stein's Algorithm**: 
   - Description: Another efficient algorithm using subtraction and division.

## Complexity Comparison Table
| Approach                  | Time Complexity          | Space Complexity         |
|--------------------------|--------------------------|--------------------------|
| Euclidean Algorithm      | O(log(min(a, b)))        | O(1)                     |
| Prime Factorization      | O(√N log N)              | O(N) (for storing factors)|
| Binary GCD Algorithm     | O(log(min(a, b)))        | O(1)                     |
| Stein's Algorithm        | O(log(min(a, b)))        | O(1)                     |
